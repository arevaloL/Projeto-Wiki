Sobre

Este projeto é uma demonstração de como criar uma página web simples usando HTML e CSS.
Ele serve como um exemplo de estrutura básica de um site, com links para diferentes seções e informações sobre séries e filmes populares.
O objetivo é fornecer uma base para quem deseja aprender mais sobre desenvolvimento web e como criar páginas informativas.
O projeto pode ser expandido com mais conteúdo, estilos e funcionalidades conforme necessário.
Para mais informações, consulte a documentação do HTML e CSS ou explore tutoriais online sobre desenvolvimento web.
